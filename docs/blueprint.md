# **App Name**: Khan Motor

## Core Features:

- Vehicle Gallery: Display a gallery of vehicles with key information.
- Vehicle Filtering: Provide filtering and sorting options for car listings.
- Contact Form: Offer a contact form to allow users to submit inquiries. Each submission will create an email to the admin.
- AI-Powered Reply Suggestions: AI tool to assist in writing personalized responses to common user inquiries, with options for customizing tone and detail before sending. The tool will reason about what to include in the output.
- Theme Toggle: Allow users to customize their display preferences with a theme toggle, supporting both light and dark themes using shadcn/ui.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to convey trust and reliability.
- Background color: Dark navy blue (#1A202C) for a modern, premium feel in dark mode. Light gray (#F9FAFB) for light mode.
- Accent color: Analogous light blue (#0FB3DD) for highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean, modern, readable style.
- Use consistent, minimalist icons from 'Font Awesome' or 'Material Icons'.
- Responsive design using a grid layout, ensuring content adapts to various screen sizes.
- Subtle transition animations for page navigation and UI elements.
